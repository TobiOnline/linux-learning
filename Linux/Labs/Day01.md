# Linux Day 1

## Objective

Learn basic Linux navigation and file management commands.

---

## Commands Learned

### pwd
Prints the current working directory.

Example:
```bash
pwd
```

---

### ls
Lists files and directories.

Example:
```bash
ls
```

---

### cd
Changes the current directory.

Example:
```bash
cd Linux
```

---

### mkdir
Creates a new directory.

Example:
```bash
mkdir Practice
```

---

### touch
Creates a new empty file.

Example:
```bash
touch notes.txt
```

---

## What I Practiced

- Navigated directories using `cd`
- Listed files with `ls`
- Created folders with `mkdir`
- Created files with `touch`

---

## Reflection

Linux commands are straightforward once you understand what each one does. Navigation and file creation are the foundation of working in a Linux environment.
